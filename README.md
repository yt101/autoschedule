# autoschedule

An application that launches multiple browser windows sequentially with configurable delays.

## Features

- Open multiple URLs in separate browser windows
- Configurable delay between window openings
- Simple, clean UI for entering URLs and settings

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

1. Enter URLs (one per line) in the textarea
2. Set the number of windows to open
3. Set the delay between window openings (in seconds)
4. Click "Launch Windows" to start

Note: Your browser may block popups. If windows don't open, check your browser's popup blocker settings.
