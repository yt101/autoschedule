# autoschedule / SesameTab

An application that launches multiple browser windows sequentially with configurable delays. Now powered by Next.js with a beautiful landing page.

## Features

- Open multiple URLs in separate browser windows
- Configurable delay between window openings
- Beautiful SesameTab landing page with marketing content
- Next.js app with Tailwind CSS styling
- Simple, clean UI for entering URLs and settings

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Start production server:
```bash
npm start
```

## Usage

1. Visit the landing page at `/` to learn about SesameTab
2. Navigate to `/app` to use the scheduler
3. Enter URLs (one per line) in the textarea
4. Set the number of windows to open
5. Set the delay between window openings (in seconds)
6. Click "Launch Windows" to start

Note: Your browser may block popups. If windows don't open, check your browser's popup blocker settings.
